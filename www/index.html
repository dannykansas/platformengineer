Congrats. Vagrant is up.

Now let's run through some very basic exercises to branch, edit, commit, and finally merge your changes using git. Make sure you have a partner to work with. Both of you should work on one system.

Task One (Teammate #1)

Go ahead and create a new branch with git, then open this file (www/index.html) and change the "bgcolor" attribute of this page from "#F6F6F6" to "#81BEF7".

When you've made that change, save the file and use "git commit" to commit the modified file to your new branch.

Now refresh the page in your browser to see your changes.

Task Two (Teammate #2)

Create another new branch, and edit the following line to include your e-mail address.

My email is: _______@something.com

Save "index.html", commit the change to your branch, and refresh the page to confirm your changes.

Task Three (Teamwork)

Switch to branch "master" and merge in Teammate #1's branch. Save, commit, confirm.

Do the same for Teammate #2's branch, and notice how both sets of code (the background changes and the added email address) now live in perfect harmony.
